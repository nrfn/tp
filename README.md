# アプリ TonePlay について
## 概要
このアプリはWEBブラウザで遊べるピアノアプリです。<br>
主な機能は二つあります。<br>
① 鍵盤に書いてあるキーを押すとその音がでます。（鍵盤右下のスイッチをクリックでON/OFFの切り替え）<br>
② 黒板に文字を入力してPLAYボタンを押すと音が出ます。<br>
(入力する文字は【SOUND CODE】や【SAMPLE MELODY】を参考にして下さい。)<br>
## 本番環境
